# 📊 Student Performance Analysis

## 📖 Overview

This project explores the key factors that influence students' academic performance using Exploratory Data Analysis (EDA).

---

## 🎯 Objectives

* Identify strong and weak relationships
* Understand key factors affecting performance

---

## 📊 Correlation Heatmap

![Correlation](images/corrplot_full.png)

---

## 🔍 Key Insights

* Failures have the strongest negative impact
* Parental education has a slight positive effect
* Study time shows weak influence
* Age has an indirect negative effect

---

## 🛠️ Tools Used

* R
* dplyr
* corrplot
